# Ứng dụng dịch văn bản

## Cài đặt thư viện

`pip install -r backend\requirements.txt`

`pip install -r frontend\requirements.txt`

## Chạy ứng dụng

Chạy backend:

`cd backend`

`python server.py`

Chạy frontend:

`cd frontend`

`streamlit run ui.py`

## Xem trực quan hóa mô hình

`streamlit run app.py`
